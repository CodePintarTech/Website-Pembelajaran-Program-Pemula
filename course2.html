Kursus JavaScript Lanjutan

Deskripsi Kursus

Kursus ini dirancang untuk mengasah keterampilan JavaScript Anda dengan teknik-teknik lanjutan dan praktik yang lebih mendalam. Anda akan belajar tentang konsep-konsep kompleks dan teknik pemrograman yang canggih.

Tujuan Kursus

  • Menguasai teknik lanjutan dalam JavaScript
  • Memahami pemrograman berbasis objek
  • Mempelajari asynchronous programming dan API

Modul Kursus

  • Modul 1: Fungsi dan Penutupan (Closures)
  • Modul 2: Pemrograman Berbasis Objek (OOP)
  • Modul 3: Asynchronous Programming (Promises & Async/Await)
  • Modul 4: Menggunakan API dan Fetch API
  • Modul 5: Latihan Proyek JavaScript
